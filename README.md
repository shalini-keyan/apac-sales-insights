# ANZ Sales Insights

Weekly sales signals hub for the ANZ Enterprise team — inspired by [@hannah-harrington/weekly-sales-insights](https://github.com/hannah-harrington/weekly-sales-insights).

**Live site:** [apacinsights.quick.shopify.io](https://apacinsights.quick.shopify.io)

---

## What it does

Every Monday morning:
1. Processes priority accounts + Demandbase intent signals into per-rep dashboards
2. Deploys a live quick site at `apacinsights.quick.shopify.io?seller=rep_name`
3. Sends each rep a personalised Slack DM with their top signals and a link to their dashboard

## Weekly run

```bash
./deploy.sh
```

This runs the full pipeline:
1. Generates `data/reps.json` from CSVs
2. Deploys the site to `apacinsights.quick.shopify.io`
3. Sends Slack DMs to all reps

### To test without sending

```bash
# Dry run all reps
python3 send-weekly-dms.py --dry-run

# Dry run for one rep
python3 send-weekly-dms.py --dry-run --rep shane_kilgour
```

### To send to one rep only

```bash
python3 send-weekly-dms.py --rep chachi_apolinario
```
